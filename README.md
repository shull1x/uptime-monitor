# Uptime Monitor 📡

Минималистичный мониторинг сайтов на Python. Скрипт раз в минуту (или как настроишь) проверяет доступность твоих ресурсов и присылает уведомление в Telegram, если сайт "лег".

Никаких сложных систем мониторинга вроде Zabbix или Grafana — просто скрипт, который делает свою работу.

## Как запустить

1. **Создай бота** через `@BotFather` и получи токен.
2. **Узнай свой Chat ID** (можно через бота `@userinfobot`).
3. **Установи зависимости**:
   ```bash
   pip install requests
   ```
4. **Настрой конфиг**:
   Открой `config.py` и впиши свои данные и список сайтов для проверки.
5. **Запусти**:
   ```bash
   python main.py
   ```

## Особенности
- **Умные уведомления:** Бот не спамит сообщениями каждую минуту. Он пишет только дважды: когда сайт упал и когда он снова поднялся.
- **Модульность:** Весь код разбит на 4 файла (логика проверки, уведомления, настройки и запуск), так что в нем легко разобраться.
- **Легкость:** Почти не потребляет ресурсы, можно запустить на самом дешевом VPS или даже на Raspberry Pi.

## Что внутри
- `checker.py` — делает HTTP-запросы и проверяет статус-коды.
- `notifier.py` — отвечает за связь с Telegram API.
- `config.py` — здесь лежат все твои ссылки и ключи.
- `main.py` — главный цикл, который все это связывает.

*Проект написан для личного пользования, пользуйтесь на здоровье!*
